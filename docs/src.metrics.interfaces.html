Evaluate the metric

Either one of evaluations or estimations must be a numpy array.

Parameters

  • evaluations_blackbox_function (np.ndarray ,optional) – Evaluations of the blackbox function. Each entry represents a single evaluation.

  • estimations_of_parameter (np.ndarray ,optional) – Parameter estimations corresponding to the blackbox evaluations. Each entry represents a single parameter estimation.

plot(evaluations_blackbox_function_for_each_experiment: Optional[Dict[Experiment, ndarray]] = None, estimations_of_parameter_for_each_experiment: Optional[Dict[Experiment, ndarray]] = None, baseline: Union[str, ndarray] = 'min') → Figure[source]#

Plot the results of the metric applied to multiple experiment and its evaluations

Parameters

  • evaluations_blackbox_function_for_each_experiment (Dict[Experiment, np.ndarray], optional) – Evaluations of the blackbox function for each experiment . Each entry represents a single evaluation.

  • estimations_of_parameter_for_each_experiment (Dict[Experiment, np.ndarray], optional) – Parameter estimations for each experiment corresponding to the blackbox evaluations. Each entry represents a single parameter estimation.

  • baseline (Union[str, np.ndarray]) – Line which should be plotted in addition to the evaluations of the metric. The dimension needs to be Line[parameter dimension][number of experiment] Current options are: - “min” (default; i.e. the minimum of all evaluated metrics) - “max” (i.e. the maximum of all evaluated metrics) - self defined line in form of numpy array with consisting of entries corresponding to the experiment - None which results in a zero baseline
